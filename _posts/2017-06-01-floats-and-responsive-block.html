In my professional activity I often face the case when I need to create two blocks: the one with fixed width floated to left or right side and the other one that must get all free space at the same row.

Let's create container block that will keep the red block with fixed with and the green block that has responsive width.
It can be done the following way

{% highlight css %} .red { width: 100px; float: left; } .green { display: block; overflow: hidden; } {% endhighlight %}

And we shouldn't forget about 'clearfix' also.

{% highlight css %} .clearfix { content: ''; display: block; clear: both; height: 0; line-height: 0; } {% endhighlight %}

Now if you resize your browser window you see that green block changes it's width while the red one stays 100px wide.

The same way we can do for right side.

{% highlight css %} .red { width: 100px; float: right; } .green { display: block; overflow: hidden; } {% endhighlight %}

So simple =)

You can add margin to fixed block if you need.

{% highlight css %} .red { width: 100px; float: right; margin-left: 24px; } {% endhighlight %}

As a bonus we will look at the case when we have two fixed width blocks on sides and the responsive one in the center.

In this case the order of blocks is valuable. First we need to place right block, then the left one and then the center one.
