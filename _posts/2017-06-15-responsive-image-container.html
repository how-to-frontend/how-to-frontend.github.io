Sometimes we have to create responsive container for image and it has to have constant propotions while we resize the browser window for we are able to place some other elements inside it.
It is pretty simple.

The steps are:

1. Create the container. Give it position relative.

{% highlight html %}
{% endhighlight %}
{% highlight css %} .image-container { position: relative; }{% endhighlight %}

2. Put the spacer empty block inside the container and give it padding top or bottom in percents equal to image's height divided by image's width.
Our example image is 1280x1024 px. So padding (I always choose padding-top) should be
1024 / 1280 * 100 = 80%.
The trick here is that vertical padding given in percents is counted not from parent's height, but from parent's width.

Because the spacer block doesn't have a content it would be semantically right to create 'span' instead of 'div'.

{% highlight html %}
{% endhighlight %}
{% highlight css %} .image-container__spacer { display: block; padding-top: 80%; }{% endhighlight %}

3. And the last step: put the absolutely positioned image inside the container.

{% highlight html %}
{% endhighlight %}
{% highlight css %} .image-container__image { position: absolute; top: 0; left: 0; width: 100%; }{% endhighlight %}
